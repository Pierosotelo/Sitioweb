CSS

¿Qué es un CSS?

El lenguaje de programación CSS o Hoja de estilo en Cascada es aquel que se utiliza para poder describir la presentación de un documento escrito en un lenguaje de marcado siendo así teniendo cosas iguales qie el HTML, siendo así que en este caso CSS se encarga de la apariencia visual de la página web, siendo así que este controla el diseño de la creación, los colores, la fuente y sobre todo la disposición de los elementos.

¿Por qué CSS se llama "Hoja de estilo en Cascada"?

Esto se llama así por la misma razón que cuando hay varios estilos aplicados todo al misml elemento lo cual gracias a esto CSS siguen una serie de reglas prioridades para así esto elija cual estilo se llegara a utilizar para así este poder ejecutarse de una buena manera.

¿Que se puede hacer con CSS?
  • Cambiar el fondo y tambien los colores.
  • Modificar el tipo, tamaño y sobre todo el estilo de letra.
  • Añadir márgenes.
  • Dividir la página en secciones y columnas.
  • Alinear textos e imagenes.
  • Hacer diseño adaptable a todos los dispositivos moviles.
  • Aplicar animaciones y transiciones para poder interactuar con el usuario.
  • Implementación para más creaciones como menús, tarjeta y botones atractivos.
